# [CryptReserve](https://www.blockchainbpi.com)
CryptReserve is a server exposing IPFS using Blockchain for permissioning and exposed vias a RestAPI.


## CryptReserve Installation Script
Run install.sh (./install.sh)

### Dependenices
* Linux Server (AWS)
* IPFS
* C++
* Json-lib
* Jansson
* libulfius-dev
* OpenSSL
* LibOrcania-dev
* Libjwt
* Hoel
* glewlwyd
* libgnutls28-dev
* JWT
